Introducción

HTML

HTML (Hyper Text Markup Language) es el lenguaje de marcado estándar para crear páginas Web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) .
Los elementos HTML son los bloques de construcción de las páginas HTML y están representados por etiquetas. Las etiquetas HTML marcan partes de contenido como "heading", "title", "body", etc.
El propósito de un navegador web (Chrome, IE, Firefox, Safari) es leer documentos HTML y mostrarlos. Los navegadores no muestran las etiquetas HTML, sino que las utilizan para procesar el contenido de la página.

HTML sigue mejorando y aparecen nuevas versiones:

Versión Año Detalles
HTML 1991 La primera propuesta oficial para convertir HTML en un estándar.
HTML 2.0 1995 La primera versión oficial de HTML el IETF publicó el estándar en septiembre de 1995.
HTML 3.2 1997 Incorpora los applets de Java y texto alrededor de las imágenes.
HTML 4.0 1998 Entre las novedades que presenta se encuentran las hojas de estilos CSS y la posibilidad de incluir pequeños programas en las páginas web.
HTML 4.01 1999 Es la última especificación oficial de HTML. Es una actualización de la versión anterior. En ese momento el W3C detuvo la actividad de estandarización de HTML hasta 2007, momento en que se retoma debido a la fuerza de las empresas que forman el grupo  WHATWG y a la publicación de los borradores de HTML 5.0, que será la siguiente versión de este lenguaje.
XHTML 1.0 2000 Es una adaptación de HTML 4.01 al lenguaje XML, por lo que mantiene sus características, y añade algunas restricciones y elementos de XML.
XHTML 1.1 2000 Ha sido publicada en forma de borrador y pretende modularizar XHTML.
XHTML 2.0 2000 Presenta grandes novedades respecto de las anteriores versiones.
HTML5 2014 Especifica dos variantes de sintaxis para HTML: una clásica, HTML (text/html), conocida como HTML5, y una variante XHTML conocida como sintaxis XHTML5 que deberá servirse con sintaxis XML. Establece una serie de nuevos elementos y atributos.
En mi manual voy a basarme sobretodo en la versión HTML5.

Estructura de la página

Etiquetas HTML

Las etiquetas HTML son nombres de elementos rodeados por corchetes angulares. Normalmente vienen en parejas como <p> y < /p> y el contenido va entre ellos.
La primera etiqueta de un par es la etiqueta de inicio, la segunda etiqueta es la etiqueta de fin. Esta última se escribe como la etiqueta de inicio, pero con una barra diagonal insertada antes del nombre de la etiqueta.

Todos los documentos HTML constan de elementos HTML anidados. Cuando se anidan etiquetas, se cierran las etiquetas en el orden opuesto al que las abrieron.
Por ejemplo si se necesita una frase en negrita, agrega abrir y cerrar etiquetas en negrita antes y después del texto. Si se necesita que sea también cursiva, la etiqueta de apertura y cierre en cursiva se agrega antes y después del bloque de texto completo. Leyendo de izquierda a derecha, se abren las etiquetas: i, b.

<i><b>Una frase</b></i>

Etiquetas de comentario HTML

Se puede agregar comentarios a su origen HTML utilizando la siguiente sintaxis: <!-- Escriba sus comentarios aquí -->
Los comentarios no se muestran en el navegador, pero pueden ayudar a documentar el código fuente HTML. La etiqueta de fin se escribe como la etiqueta de inicio, pero con una barra diagonal insertada antes del nombre de la etiqueta.

Estructura del documento

<!DOCTYPE html>  <!-- Declaracción que define este documento como HTML -->
<html>  <!-- Inicio -->
   <head>  <!-- Cabecera -->
   </head>
   <body>  <!-- Cuerpo -->
   </body>
</html>  <!-- Fin -->

La declaración <!DOCTYPE> representa el tipo de documento y ayuda a los navegadores a mostrar páginas web correctamente. Solo debe aparecer una vez, en la parte superior de la página (antes de cualquier etiqueta HTML).

Etiqueta html
<html lang="es">

El documento HTML comienza con <html> y termina con </ html>. Es el elemento raíz de una página HTML.
El idioma del documento se puede declarar en esta etiqueta y se declara con el atributo lang.

Etiqueta head
<head>
<meta charset="utf-8">
<meta name="description" content="HTML">
<meta name="author" content="Javier Alonso">
<meta name="keywords" content="HTML">
<link rel="icon" href="favicon.ico">
<title>Lenguaje de marcas - HTML</title>
<link href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css" rel="stylesheet">
<style>
  h1{
      margin-bottom: 30px;
  }
</style>
</head>

El elemento <head> es un contenedor para metadatos. Los metadatos HTML son datos sobre el documento HTML y no se muestran.
Los siguientes describen las etiquetas de metadatos:

  1. meta
  2. title
  3. link
  4. style
  5. script
  6. base

Nombre Detalles
meta El elemento meta se utiliza para especificar qué conjunto de caracteres se utiliza, de descripción de páginas, palabras clave, autor, y otros metadatos.
  • La etiqueta meta charset se utiliza para visualizar una página HTML correctamente. Un navegador web debe conocer el conjunto de caracteres utilizado en la página. "UTF-8" (Unicode) se utiliza en HTML5 y tiene casi todos los caracteres y símbolos en el mundo.
  • Se puede definir una descripción de una página web con name="description" en sección content.
  • Se puede definir el autor de una página en content con name="author".
  • Definir palabras clave para motores de búsqueda en content con name="keywords".
title La etiqueta title especifica un título para el documento.
link El elemento link se utiliza para añadir un elemento externo.
  • Para vincular a las hojas de estilo externas con rel="stylesheet" y href con enlace al fichero con estilo css. Es popular y muy rápido usar listas de estilos de código abierto (bootstrap) desde las páginas como por ejemplo bootstrapcdn.com
  • Para mostrar también un icono, se utiliza link rel="icon" href="favicon.ico".
style Define la información de estilo interno para un documento.
script El elemento script se utiliza para definir archivos JavaScript del lado del cliente.
base El elemento base especifica la dirección URL base y destino de base para todas las URLs relativas en una página.

Gracias a title y link se puede ver el titulo de la pagina en un navegador como:



Etiqueta body
<body>
   <h1>Encabezado</h1>  <!-- Define un encabezado grande -->
   <p>Párrafo</p>  <!-- Define un párrafo -->
</body>
</html>

La parte visible del documento HTML está entre <body> y </ body>. Contiene todo el contenido de un documento HTML, tales como texto, hipervínculos, imágenes, tablas, etc.

Secciones

HTML5 ofrece nuevos elementos semánticos para definir diferentes partes de una página web. Son los elementos semanticos que defienen claramente su contenido:



Article y section

La etiqueta <article> especifica contenido independiente y autónomo. Un artículo debe tener sentido por sí mismo y debe ser posible distribuirlo independientemente del resto del sitio.

<article>
<header>Encabezado</header>
<p>El texto del articulo</p>
</article>
El elemento <header> especifica un encabezado para un documento o sección y debe usarse como un contenedor para contenido introductorio.
La etiqueta <section> define secciones de un documento, como capítulos, encabezados, pies de página o cualquier otra sección del documento.
<section>
<h1>Título</h1>
<p>El texto del articulo</p>
</section>
No hay consenso a la hora de como anidar esos elementos. En Internet se pueden encontrar páginas HTML con elementos <section> que contienen elementos <article> y <article> que contienen elementos <section>. Normalmente, esta es la forma más correcta:
<article>
   <section>
   </section>
   <section>
   </section>
</article>
...aunque esta es común también:
<section>
   <article>
   </article>
   <article>
   </article>
</section>

Footer

El elemento <footer> especifica un pie de página para un documento o sección. Un pie de página normalmente contiene al autor del documento, información sobre derechos de autor, enlaces a términos de uso, información de contacto, etc.

<footer>
   <p>Una información sobre el autor</p>
</footer>

Nav

El elemento <nav> define un conjunto de enlaces de navegación.

<nav>
   <a href="http://www.google.com">Google</a>
   <a href="http://www.heraldo.es">Heraldo</a>
</nav>

Etiqueta <div>

La etiqueta <div> define una división o una sección en un documento HTML y se utiliza para agrupar elementos de bloque para darles formato con estilo. Es un ejemplo de elemento no semántico - no dice nada sobre su contenido.

<div style="width: 800px; margin: 5px; color: blue; background-color: yellow; border: 1px solid black;" >
<h2 >Elemento agrupado 1</h2>
<p>Elemento agrupado 2</p>
</div>
En este ejemplo estilo <style> de etiqueta <div> define una anchura width en pixels, margen margin en pixels, color de la fuente color, color del fondo background-color y un borde border.

Elementos

Encabezado

Los encabezados HTML se definen con las etiquetas <h1> a <h6>.
<h1> define el encabezado más importante y <h6> define el encabezado menos importante.

Párrafos

Los párrafos HTML se definen con la etiqueta <p>

<p>Un párrafo</p>

Texto preformateado

Los párrafos HTML se definen con la etiqueta <pre>. Conserva los dos espacios y saltos de línea. 

Puede ser muy útil
						en algunos casos
<pre>Un texto preformateado</pre>

Enlaces

Los enlaces HTML se definen con la etiqueta <a>.

<a href="http://www.google.com" title="Un título de enlace">Un enlace</a>
El destino del enlace se especifica en el atributo href.
Se puede utilizar los atributos adicionales para proporcionar información sobre elementos HTML, como por ejemplo el título del enlace title.

Tipos de enlaces

Hay dos tipos: absoluto y relativo.
Un enlace absoluto contiene toda la dirección de la pagina como http://www.google.com.
Un enlace relativo se divide en tres diferentes tipos:

  1. Mismo directorio, por ejemplo href="index.html"
  2. Subdirectorio, por ejemplo href="carpeta/autor.html"
  3. Directorio anterior, por ejemplo href="../autor.html"

Punto de anclaje

Se puede crear una referencia a otro sitio en la misma página:

<a name="nombreDeAnclaje" >Otro sitio</a>
Al pinchar en un enlace con atributo href el navegador muestra ese sitio en la pagina:
<a href="#nombreDeAnclaje" >Otro sitio</a>

Imágenes

Las imágenes HTML se definen con la etiqueta <img>.

<img src="imagen.jpg" alt="Un texto alternativo de imagen" width="250" height="100" >
La ruta del archivo fuente se acede con src.
Se puede utilizar los atributos adicionales como el texto alternativo alt que se muestra si no se puede mostrar la imagen, el ancho width y la altura height.

Etiqueta <br>

La etiqueta <br> inserta un salto de línea único. Es una etiqueta vacía que significa que no tiene etiqueta de final.
En HTML, la etiqueta no tiene etiqueta de final. En XHTML (la versión más nueva) la etiqueta debe estar correctamente cerrada: <br />.

<br>

Etiqueta <wbr>

La etiqueta <wbr> (word break opportunity) especifica dónde en un texto que estaría bien añadir un salto de línea.
Por ejemplo cuando una palabra es demasiado larga para evitar situacción cuando el navegador va a romper sus líneas en el lugar equivocado, se puede utilizar wbr para signalizar donde dividir las palabras.

<p>El texto demasiado largo <wbr> separado con wbr</p>

Etiqueta <hr>

La etiqueta <hr> se usa para para separar el contenido o definir un cambio temático en una página HTML.

<hr>

Listas

Listas desordenadas

Una lista sin orden comienza con la etiqueta <ul>. Cada elemento de la lista comienza con la etiqueta <li>.

<ul style="list-style-type:circle">
<li>Elemento1</li>
<li>Elemento2</li>
<li>Elemento3</li>
</ul>
Los elementos de la lista se marcarán con balas (pequeños círculos negros) por defecto. Se puede cambiar el marcador con la propiedad list-style-type.

Listas ordenadas

Una lista sin orden comienza con la etiqueta <ol>. Cada elemento de la lista comienza con la etiqueta <li>.

<ol type="a">
<li>Elemento1</li>
<li>Elemento2</li>
<li>Elemento3</li>
</ol>
Los elementos de la lista se marcarán con números de forma predeterminada. Se puede cambiar el marcador con la propiedad type:
Nombre Descripción
1 Los elementos de la lista se numerarán con números.
a Los elementos de la lista se numerarán con letras minúsculas.
A Los elementos de la lista se numerarán con letras mayúsculas.
i Los elementos de la lista se numerarán con números romanos en minúsculas.
I Los elementos de la lista se numerarán con números romanos en mayúsculas.
Para obtener el orden descendente de la lista se utiliza reversed:
<ol reversed>
<li>Elemento1</li>
<li>Elemento2</li>
<li>Elemento3</li>
</ol>
Ejemplo:
  1. Elemento1
  2. Elemento2
  3. Elemento3
Para especificar el valor inicial para numerar los elementos individuales de la lista se utiliza start:
<ol start="3">
<li>Elemento1</li>
<li>Elemento2</li>
<li>Elemento3</li>
</ol>
Ejemplo:
  1. Elemento1
  2. Elemento2
  3. Elemento3

Definition list

Es una lista de términos, con una descripción de cada término. Se comienza con la etiqueta <dl>. La etiqueta <dt> define el término (nombre) y la etiqueta <dd> describe cada término.

<dl>
<dt>Definición1</dt>
<dd>- explanación1</dd>
<dt>Definición2</dt>
<dd>- explanación2</dd>
</dl>
Por ejemplo:
ul
- unordered list
ol
- ordered list
dl
- definition list

Tablas

Una tabla HTML se define con la etiqueta <table>.
Las etiquetas de la tabla HTML son:

Nombre Descripcción
thead Engrupa el contenido de la cabecera de una tabla.
tbody Engrupa el contenido del cuerpo de una tabla.
tfoot Engrupa el contenido del pie de una tabla.
caption Define un título de la tabla.

Cada fila de la tabla se define con la etiqueta <tr>, un encabezado de tabla se define con la etiqueta <th> una celda de datos con la etiqueta <td>.
<table style="border: 1px solid black;">  <!-- El estilo define un borde de la tabla -->
<caption>El título de la tabla</caption>
<thead>
<tr>
<th>Título1</th>
<th>Título2</th>
<th>Título3</th>
</tr>
</thead>
<tbody>
<tr>
<td>Celda1</td>
<td>Celda2</td>
<td>Celda3</td>
</tr>
</tbody>
<tfoot>
<tr>
<td>Celda3</td>
<td>Celda4</td>
<td>Celda5</td>
</tr>
</tfoot>
</table>
Expansiones

Se puede hacer una expansión de una columna para hacer un espacio intencionado abarcando los siguientes espacios de columnas próximas.

<table>
<tr>
<th>Título1</th>
<th colspan="2">Título2</th>
</tr>
<tr>
<td>Celda1</td>
<td>Celda2</td>
<td>Celda3</td>
</tr>
</table>
Título1 Título2
Celda1 Celda2 Celda3
También se puede hacer una expansión en una fila:
<table>
<tr>
<th>Título1</th>
<td>Celda1</td>
</tr>
<tr>
<th rowspan="2">Título2</th>
<td>Celda2</td>
</tr>
<tr>
<td>Celda3</td>
</tr>
</table>
Título1 Celda1
Título2 Celda2
Celda3

Resaltado

i - Italic

La <i> especifica el texto en cursiva.

<i>El texto en cursiva</i>

b - Bold

La <b> especifica el texto en negrita.

<b>El texto en negrita</b>

u - Underlined

La <u> especifica el texto subrayado.

<u>El texto subrayado</u>

em - Stress emphasis

La etiqueta <em> es una etiqueta frase. Se hace hincapié en forma de texto.

<em>El texto con énfasis</em>

strong

La <strong> define el texto importante.

<strong>El texto importante</strong>

small

La <small> especifica el texto más pequeño.

<small>El texto más pequeño</small>

s

La <s> especifica el texto que ya no es correcto, exacto o pertinente.

<s>El texto ya no correcto</s>

Editores

  • Atom
  • Brackets

CSS

CSS (Cascading Style Sheets) es un lenguaje para describir páginas web de presentación (visualización). Fue diseñado por W3c en 1996.

JavaScript

Javascript es un lenguaje de programación de secuencias de comandos creado por Netscape en 1995. Script puede proporcionar interactividad a través de la reacción en los eventos, la comprobación de la validación de los formularios, etc.
